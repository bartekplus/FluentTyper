Thank you for installing FluentTyper

FluentTyper attempts to improve the ease and speed of textual input by predicting words. It autocompletes words for you while you typing. FluentTyper will also fix your typos.

FluentTyper in action

Cheat sheet

  • Reload all open web pages to activate the extension
  • To disable the extension for a website, click the FluentTyper icon in the browser toolbar
  • Start typing, FluentTyper will show the popup with the list of suggested words
  • Use the arrow keys to select the word from the list of suggested words
  • Use Enter, Tab, or Space key to confirm the selection
  • Use the Escape key or just continue typing to hide the list of suggested words
  • Use the Backspace key to undo autocorrection
