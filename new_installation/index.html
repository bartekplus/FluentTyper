FluentTyper

Thank you for installing FluentTyper

FluentTyper attempts to improve the ease and speed of textual input by predicting words. It autocompletes words for you while you typing. FluentTyper will also fix your typos.

FluentTyper in action

Cheat sheet

  • Start typing, FluentTyper will show popup with the list of suggested words
  • Use the arrow keys to select word the list of suggested words
  • Use Enter or Tab key to confirm selection
  • Use Escape key or just continue typing to hide the list of suggested words
